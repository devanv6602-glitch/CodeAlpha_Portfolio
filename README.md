# Personal Portfolio Website

A premium, responsive personal portfolio built with HTML, CSS, and JavaScript only. This project is optimized for internship applications and showcases modern frontend techniques.

## Features

- Sticky navigation bar with responsive hamburger menu
- Smooth scrolling and section highlighting
- Typing animation in the hero section
- Animated skill progress bars
- Project cards with GitHub and Live Demo buttons
- Dark / light theme toggle with local storage
- Scroll-to-top button
- Loading animation and scroll reveal effects
- Glassmorphism design with gradient backgrounds
- Semantic HTML, modern CSS Grid/Flexbox layout, and clean JavaScript interactions

## Project Structure

- `index.html` — main website markup
- `style.css` — responsive styling and animations
- `script.js` — navigation, theme switch, animations, form handling
- `assets/images/` — image and asset folder structure
- `README.md` — project overview and instructions

## Usage

1. Open `index.html` in a browser.
2. Customize the content, contact information, and project links.
3. Replace placeholder assets as needed for a polished internship submission.
